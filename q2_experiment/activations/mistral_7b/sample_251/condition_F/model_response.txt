Here's a simple function in Python to convert Fahrenheit to Celsius:

```python
def fahrenheit_to_celsius(fahrenheit):
    celsius = (fahrenheit - 32) * 5 / 9
    return celsius
```

This function takes a temperature in Fahrenheit as an argument, calculates the equivalent temperature in Celsius using the formula `(Fahrenheit - 32) * 5 / 9`, and then returns the result as a floating point number.

Example usage:

```python
print(fahrenheit_to_celsius(32))
